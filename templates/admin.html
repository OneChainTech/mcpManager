MCP 配置

在客户端的 mcp.json 中添加如下配置，指向本服务的 /mcp 路径：

{
  "mcpServers": {
    "api-proxy-mcp": {
      "url": "http://127.0.0.1:8000/mcp"
    }
  }
}
  • 确保本服务已启动：访问 /health 返回 {"status":"ok"}。
  • 根据实际部署修改端口或域名。
  • 保存后重启你的 MCP 客户端/编辑器。
